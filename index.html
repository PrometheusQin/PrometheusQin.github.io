本地博客网站建立流程

1. 安装git

windows下安装msysGit即可

2. 安装Node.js

下载安装包安装

3. 安装Hexo

4. 创建hexo文件夹

在一特定文件夹（D:\hexo),右键执行Git Bash，键入命令：hexo init,hexo会自动在目标文件夹生成建立网站所需要的文件

5. 安装依赖包

npm install

6. 本地查看

此刻已经搭配好本地的hexo博客，执行如下命令

到浏览器中输入：localhost：4000查看
至此，本地博客搭建完成。
